# gitmanage

## Overview
gitmanage is a simple Task Tracker used to demonstrate Git and GitHub workflows across four phases.

## Installation
1. Clone: `git clone git@github.com:<your-username>/gitmanage.git`
2. (Optional for Python) Create venv: `python -m venv venv && source venv/bin/activate`
3. Run: `python src/main.py`

## Features
- Basic CLI demo app
- Demonstrates branching, merging, and conflict resolution
- GitHub Actions CI workflow
- Documentation published using GitHub Pages

## Usage
Run `python src/main.py` to test a simple output.

## Security & Ethics Guidelines

### 1. Do not commit secrets
Never upload API keys, passwords, tokens, or database credentials.

### 2. Avoid personal or sensitive data
Do not commit real customer information, private files, or large datasets.
Use fake/synthetic/redacted data for examples.

### 3. Do not send confidential data to AI tools
Never paste real production data into ChatGPT, Copilot, or any AI tools.

### 4. Use synthetic/redacted samples
Replace sensitive data with placeholders or fake examples.

### 5. Version control ensures accountability
Git keeps full history, so commit responsibly and avoid sensitive info.
